# 🎟️ Bookify – MERN Event Booking Platform

Bookify is a full-stack **MERN** application that allows users to browse, book, and manage events seamlessly. Designed for scalability and ease of use, Bookify offers role-based access for users and admins with features such as authentication, real-time updates, booking system, payments, and more.

---

## 🚀 Features

### 👥 Authentication & Roles
- Secure **JWT-based authentication**
- Two user roles: `User` and `Admin`
- Role-based access control

### 📅 Event Management
- Admins can **create**, **edit**, and **delete** events
- Users can **browse** and **book** events
- Events include name, date, location, ticket availability, and price

### 🛒 Booking System
- Users can book available events
- Automatic availability check
- Booking confirmation and user-specific booking history

### 💳 Payments
- Integrated **payment gateway** (e.g., Stripe)
- Real-time payment confirmation and error handling

### 🔔 Email & Notifications
- Confirmation emails for bookings
- Admins receive booking alerts
- Future support for push notifications

### 🧑‍💼 Admin Dashboard
- Event management panel
- View total bookings and user list
- Monitor revenue and booking statistics

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS, Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JWT
- **Payments:** Razorpay
- **Deployment:** Vercel (Frontend), Render / Railway (Backend)
